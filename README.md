# Classificador Binário de Doenças Cardiovasculares
As doenças cardiovasculares são a principal causa de morte em todo o mundo, tornando a detecção precoce um desafio crítico para a saúde pública.  3 O objetivo deste projeto é construir um classificador binário para prever a presença (1) ou ausência (0) de doença cardíaca em um paciente com base em um conjunto de atributos clínicos.
